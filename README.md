#[proCV](https://procv-zk.netlify.app/)  - Professional CV Builder



proCV is a modern, fast, and elegant CV builder built with React and Vite. It allows users to choose from professional templates, fill in their details with real-time validation, and preview their resume instantly.

## ✨ Features

- **Template Selection**: Choose from multiple professional design templates.
- **Real-time Validation**: Powered by **React Hook Form** and **Zod** to ensure your data is perfect.
- **Live Preview**: Instantly see how your resume looks as you fill out the information.
- **Modern UI**: Styled with **Tailwind CSS** and featuring **Lucide React** icons.
- **Print Ready**: Optimized layouts for A4 printing and PDF export.

## 🚀 Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Form Handling**: [React Hook Form](https://react-hook-form.com/)
- **Validation**: [Zod](https://zod.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router](https://reactrouter.com/)

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/components`: Reusable UI components (NavBar, TemplateDesign, etc.)
- `src/views`: Main page views (HomePage, DrawingPage, InformationForm, PreviewPage)
- `src/assets`: Image assets and template placeholders
- `src/index.css`: Global styles and Tailwind configurations
